# ACKS-Judge-s-Toolkit
ACKS Judge's Toolkit is a Windows desktop toolkit for OSR/ACKS campaign preparation. ACKS Judge's Toolkit - это программа на Windows для кампаний по OSR / ACKS.

ACKS Judge's Toolkit is a Windows desktop toolkit for OSR/ACKS campaign preparation: it generates regional hex maps, settlements, domains, realms, roads, rivers, trade data, dungeons, characters, and treasure, with editable records and Excel import/export support.

ACKS Judge's Toolkit - это программа на Windows для кампаний по OSR / ACKS:  создает региональные гексовые карты, поселения, домены, королевства, дороги, реки, спрос и предложение, подземелья, персонажей и сокровища с возможностью редактирования записей и поддержкой импорта/экспорта в Excel.
